# Contributing to Zombie Shooter

Thank you for your interest in this project! 

## Project Status: Archived / Legacy
Please note that this is a **legacy portfolio project** built during my undergraduate studies. It is currently archived and **not under active development**. 

However, because this repository serves as an educational case study for Unity development and C# scripting, contributions in the form of bug fixes, optimization examples, or documentation improvements are still welcome!

## How to Contribute

If you'd like to experiment with the code or suggest an improvement:

1. **Fork the Repository**
2. **Create a Feature Branch** (`git checkout -b feature/AmazingFeature`)
3. **Commit your Changes** (`git commit -m 'Add some AmazingFeature'`)
4. **Push to the Branch** (`git push origin feature/AmazingFeature`)
5. **Open a Pull Request**

## Reporting Issues
If you encounter a bug or issues importing the legacy assets into a modern Unity version, feel free to open an Issue. While I may not actively release patches, documenting the issue helps others who are studying the codebase.

## Code of Conduct
Please note that this project is released with a [Code of Conduct](CODE_OF_CONDUCT.md). By participating in this project you agree to abide by its terms.
